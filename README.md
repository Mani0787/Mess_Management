# Mess Management System

A web-based application designed to simplify and automate the daily activities of a hostel mess. This system enables users to register, record attendance, and track meal schedules, while administrators can manage menus, users, and billing records.

---

## Features

### User Features
- Secure Login & Registration
- View Weekly Meal Menu
- Mark Attendance for Meals
- Check Monthly Bill Summary
- Update Profile Information

### Admin Features
- Manage User Accounts
- Add/Update/Delete Meal Schedules
- Monitor Meal Attendance Logs
- Generate and Manage Billing Reports
- Dashboard for Complete Control
